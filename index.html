04Cultural Context Overlay — Expanded

Axis 3 hoạt động như filter overlay — quyết định items nào hiện/ẩn theo region. Tách riêng vì "Modest" không phải occasion (khi nào mặc) mà là giá trị (tại sao mặc thế này). Validated by ASOS (Modest = top-level) và Shein (6+ cultural categories).

Modest Fashion Overlay

Regions: MENA, Pakistan, Malaysia, Turkey, Indonesia
Existing theme: Modest / Ramadan → now overlay filter
CW Action: Filter revealing items. Boost hijab-friendly, longer hemlines, covered shoulders. Auto-inject during Ramadan/Eid
Sources: ASOS Modest = top-level edit · Shein MOD sub-brand + Arabian/Kaftan/Abayas · Love Nikki cultural tags

Traditional Formal Overlay

Regions: South Asia, East Asia, LATAM — during cultural events
Content: Saree, Áo dài, Hanbok, Hanfu, Cheongsam, Kimono, Lehenga
CW Action: Auto-inject during Diwali, Lunar New Year, Tet, Navratri...
Sources: Love Nikki 10+ cultural tags · Shein Hanfu/Cheongsam · Life Makeover: Chinese, Chinese Heritage, Chinese Imperial, Japanese, Egyptian tags

Ethnic Heritage Overlay

Regions: All — for cultural appreciation events
Content: Regional patterns, tribal prints, indigenous textiles, folk costumes, national dress
CW Action: Feature during Independence Days, Heritage Months, multicultural events
Sources: Covet's "Cultural/Ethnic" theme · Love Nikki multicultural tags · Life Makeover: British, European, Ethnic tags
🔄 How Overlays Work with Occasion Categories
Overlay KHÔNG thay thế occasion — nó filter bên trên. Ví dụ: Event "Valentine's Day" ở Saudi Arabia → pull từ Category "Night Out & Romance" → apply "Modest Fashion Overlay" → kết quả: romantic outfits nhưng covered shoulders, longer hemlines. Cùng event ở Brazil → pull cùng category → không apply overlay → kết quả: full range romantic outfits.

Per-Game Detail
Love Nikki Paper Games · 2017
Pioneer
Gameplay: Style battle PvP — attribute scores compete. ~45 thematic tags trộn occasion + cultural + fantasy vào 1 flat list. Cultural depth sâu nhất (Chinese Classical, Cheongsam, Republic of China, Modern China, Kimono, Britain...)
✓ Pros
Tag chi tiết nhất. Cultural depth vượt trội. Proven 7+ năm
✗ Cons
Flat list — no hierarchy. 50 tags quá nhiều. Trộn 3 axes
Infinity Nikki Infold Games · 2024
Best Count
Gameplay: Open-world 3D. Labels quyết định quest compatibility. 20 labels — simplified từ Love Nikki. Tách rõ occasion (Formal, Uniform) vs mood (Romance, Fantasy)
✓ Pros
20 labels = manageable. Bắt đầu tách occasion vs mood
✗ Cons
Vẫn flat. Thiếu cultural + sport + street
Life Makeover Archosaur Games · 2023
Most Tags
Gameplay: 3D open-world + detective story. Fashion dye & DIY system. Items have mandatory Style (scoring) + optional Tags (discovery). ~55 tags — richest system overall. Items also have garment Types (Dress, Coat, Top, Bottom, Shoes... + Makeup: Eyeshadow, Lipstick, Blush...). Covers all 4 axes: occasion (Wedding, Formal, Holiday, Swimsuit), style (Cool, Elegant, Gorgeous), cultural (Chinese Heritage, Chinese Imperial, Japanese, Egyptian, British, European), climate (Rain Gear, Sunproof, Winter). Fashion Studio cho crafting + Decom Bench cho recycling
✓ Pros
~63 total tags — most comprehensive. Covers all 4 axes naturally. 2-layer system (Style mandatory + Tags optional) = clean architecture. DIY/dye system adds depth
✗ Cons
~55 tags flat list — no hierarchy. Trộn 4 axes vào 1 list. Single-game design, not multi-title CW
Dress Up! Time Princess IGG · 2020
Multi-Dim
Gameplay: Story-driven historical periods (Marie Antoinette, Tang Dynasty, 1920s NYC). 3-dimension system: Style Pairs × Scene × Color. Opposed pairs reduce overlap automatically
✓ Pros
Multi-axis concept closest to our approach. Opposed pairs reduce overlap
✗ Cons
14 pairs × scenes × colors = too complex. Scene tags only for story games
Covet Fashion Crowdstar (EA) · 2013
Closest to CW
Gameplay: Real-brand shopping (150+ labels). Weekly challenges judged by real players. 16 occasion themes — pure occasion-first. Season-based progression
✓ Pros
16 themes = proven sweet spot. Pure occasion → maps to Event Master. 10+ years validated
✗ Cons
Flat list. No style axis → can't build taste profile. Cultural depth weak
Glow: Fashion Idol CrazyLabs · 2023
LiveOps-Driven
Gameplay: Theme-based missions. OOTD battles. LiveOps-heavy with 5-7 collection drops per update. Collection-based naming (Pond Fairy, Street Code, Urban Threads). F2P + ads model
✓ Pros
Collection model = LiveOps cadence. Poetic names drive aspiration. F2P + ads relevant
✗ Cons
No systematic taxonomy. Collections are ad hoc. Not scalable multi-title
